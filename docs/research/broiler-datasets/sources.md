# Sources and verification boundaries

- Crossref: DOI and deposited scholarly metadata. This is not an indexing certification.
- OpenAlex: broad scholarly catalog and open-access discovery. Validate decisive bibliographic and venue claims separately.
- Europe PMC: biomedical/life-science literature discovery. An abstract or record is not the same as reviewing full text.
- Publisher/repository landing pages: use for confirming publication status, article type, corrections/retractions, and legitimate access.

Retrieved records are candidates, not a final literature review.
